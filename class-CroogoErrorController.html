Class CroogoErrorController

Error Handling Controller

Controller used by ErrorHandler to render error views. This is based on CakePHP's own CakeErrorController with the following differences:

  • loads its own set of components and helpers
  • aware of Site.theme and Site.admin_theme

PHP version 5
