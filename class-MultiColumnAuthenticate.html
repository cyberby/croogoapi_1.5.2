Class MultiColumnAuthenticate

An authentication adapter for AuthComponent. Provides the ability to authenticate using POST data. The username form input can be checked against multiple table columns, for instance username and email

{{{ $this->Auth->authenticate = array( 'Authenticate.MultiColumn' => array( 'fields' => array( 'username' => 'username', 'password' => 'password' ), 'columns' => array('username', 'email'), 'userModel' => 'User', 'scope' => array('User.active' => 1) ) ) }}}
